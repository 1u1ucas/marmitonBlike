# marmitonBlike

marmitonBlike est une application de recettes de cuisine. Vous pouvez créer, partager et découvrir de nouvelles recettes.

## Fonctionnalités

- Créer des recettes
- Partager des recettes
- Découvrir de nouvelles recettes

## Comment l'utiliser

1. Clonez ce dépôt

### Utilisation avec Docker Compose

1. Construisez et lancez vos services avec `docker-compose up -d`
2. Arrêtez et supprimez les conteneurs avec `docker-compose down`

Notez que vous devez avoir Docker et Docker Compose installés sur votre machine pour utiliser ces commandes.
